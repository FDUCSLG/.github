# FDUCSLG

**NOTE:** the "sys-ta" team has moved to GitLab. Please check-out <https://gitlab.com/fudan-systa>.

FDUCSLG，创始于 2020 年，是一个由复旦大学学生驱动的计算机技术社区，全称为 FDU Computer Science Lovers' Group。我们秉持**提升计算机技术氛围**的理念，致力于构建一个真正属于复旦人的技术生态与开源平台。

我们与**中科大 LUG、北大 Linux 俱乐部**等友校社团保持紧密合作，并与 **JetBrains、字节 TRAE** 等企业合作举办校内活动，为你连接更广阔的技术视野。

目前，社团已汇聚了 600 多名 成员，其中 60 余名核心成员构成了坚实的中坚力量。这里是一个友善且高效的交流平台，无论你是想探讨前沿技术、寻找项目伙伴，还是单纯地想分享一个酷点子，我们都欢迎你的加入。

让我们一起，从代码开始，构建下一个可能。
